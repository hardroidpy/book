
\subsection{Alcance}
\begin{frame}{Alcance}

\framesubtitle{Propuesta}
\begin{center}
Reconocer actividades humanas utilizando teléfonos móviles modernos
con un enfoque colaborativo
\par\end{center}

\end{frame}
%
\begin{frame}{Alcance}

\framesubtitle{Objetivo General}
\begin{center}
El objetivo principal es desarrollar un sistema \structure{HAR} utilizando
teléfonos móviles modernos cuyo aporte principal es un una librería
de código abierto y un modelo activo.
\par\end{center}

\end{frame}
%
\begin{frame}{Alcance}

\framesubtitle{Objetivos Específicos}

\setbeamercovered{transparent}
\begin{enumerate}[<+->]
\item Definir el marco teórico sobre \structure{HAR}. 
\item Revisar las técnicas de \structure{captura de señales} bajo restricción
de consumo de energía. 
\item Revisar el \structure{procesamiento de señales} para identificar
variables de entrenamiento. 
\item Comprender la clasificación basada \structure{aprendizaje automático}
bajo restricción de consumo energía. 
\item Diseñar un \structure{sistema HAR} que incluya la recolección de
muestras colaborativas y clasificación de actividades en-línea. 
\item Aportar un componente de \structure{librería} para teléfonos móviles
modernos con plataforma \emph{\textbf{\emph{Android}}}. 
\end{enumerate}
\end{frame}
